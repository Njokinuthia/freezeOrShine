## Description
</p>A simple website page that allows a user to get current weather information of a city of choice.

## Installation

### Requirements

* Either a computer,phone,tablet or an Ipad

* An access to the Internet

## Technology Used
* HTML5
* CSS 
* Bootstrap
* Javascript 
* open weather


## Behaviour Driven Development

1. User input loads page.
2. Type in city they want weather data on. 
3. Submit the city input
4. Page displays information
5. Background image changes according to search result. 

## Known Bugs
* Background image at times takes long to load.
